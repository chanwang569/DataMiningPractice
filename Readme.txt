代码功能逻辑分为四个部分。算法实现模块，数据预处理模块，数据集分析模块和工具模块。算法模块封装实现的三个算法，分别命名为LR模块，BP模块，及CART模块。每个模块封装了算法的训练方法及预测方法。
数据预处理模为Jupyter脚本文件。Jupyter[8]是因其良好的交互性并能实现实时的结果展示。非常适合对数据集进行探索性预处理。每个数据集有对应的Jupyter文件，bank.ipynb包含对银行数据集进行可视化预览、属性分布及相关性分析、去除噪点、删除属性等数据预处理操作。同理，blood.ipynb是对数学中心数据集的处理，SUSY.ipynb是对SUSY数据集的初步与预处理。
针对数据集进行算法研究的数据集分析模块也是按照数据集组织的。ExploreBankDataset.py是对银行数据集的算法研究，可以通过注释或者取消注释来组合应用分类算法。同理ExploreBloodDataset.py是对血液中心数据集的研究，ExploreSUSYDataset.py是对SUSY数据集的研究。
工具模块包含几个独立的源文件。Utils文件包含可以被各个模块使用的工具函数，Result模块包含对结果进行分析和展示的工具
